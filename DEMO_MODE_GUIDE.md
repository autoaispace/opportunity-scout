# 演示模式使用指南 🎭

## ✨ 功能概述

演示模式允许您**无需配置 OAuth**即可完整体验 Opportunity Scout 的所有功能。

---

## 🚀 快速开始

### 1️⃣ 访问登录页面
```
http://localhost:3000/login
```

### 2️⃣ 点击"演示模式登录"
- 无需真实 Google/Apple 账号
- 自动创建演示用户
- 立即进入应用

---

## 🎮 完整体验流程

### 第一步：登录
1. 访问 `/login`
2. 点击 **"演示模式登录（免配置）"**
3. 自动跳转到 Feed 页面

### 第二步：浏览需求
1. 看到 5 条商机需求
2. **前 3 条**：完全可见（免费用户权限）
3. **第 4-5 条**：显示锁定状态

### 第三步：尝试解锁
1. 点击任意锁定的内容
2. 弹出定价 Modal
3. 3 个套餐可选：
   - 月付：¥99/月
   - **年付：¥999/年**（推荐，有"最超值"标签）
   - 终身：¥1999/终身

### 第四步：模拟支付
1. 选择套餐 → 跳转支付页面
2. 点击 **"完成支付（演示）"**
3. 1.5 秒延迟模拟真实支付
4. 自动跳回 Feed

### 第五步：享受 Pro 功能
1. 页面顶部显示 **"升级成功！🎉"**
2. 所有锁定内容自动解锁
3. 右上角显示 **"🔥 Pro 会员"**

## 👤 用户状态管理

### 演示用户信息
```json
{
  "id": "demo-user-123",
  "email": "demo@opportunityscout.ai",
  "name": "演示用户",
  "isPro": false  // 支付后变为 true
}
```

### 存储位置
- `localStorage.demo_user`

### 查看当前状态
```javascript
// 在浏览器控制台运行
console.log(JSON.parse(localStorage.getItem('demo_user')))
```

## 🔧 开发者工具

### 重置演示状态
```javascript
// 清除演示用户
localStorage.removeItem('demo_user')

// 降级为免费用户
const user = JSON.parse(localStorage.getItem('demo_user'))
user.isPro = false
localStorage.setItem('demo_user', JSON.stringify(user))

// 升级为 Pro 用户
const user = JSON.parse(localStorage.getItem('demo_user'))
user.isPro = true
localStorage.setItem('demo_user', JSON.stringify(user))
```

// Mock 支付会在控制台输出

## 🆚 演示模式 vs 真实 OAuth

| 功能 | 演示模式 | 真实 OAuth |
|------|---------|-----------|
| 配置难度 | ✅ 零配置 | ⚠️ 需配置 Supabase |
| 账号绑定 | ❌ 本地存储 | ✅ 真实账号 |
| 数据持久化 | ❌ 清除缓存即丢失 | ✅ 数据库存储 |
| 支付功能 | 🎭 模拟支付 | 💳 真实支付（Stripe） |
| 适用场景 | 演示、测试、开发 | 生产环境 |
